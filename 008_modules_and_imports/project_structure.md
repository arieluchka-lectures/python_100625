


1) modules = a python file

2) packages = is a collection of modules.

3) library 


### pip
to install a package:
```
pip install <package_name>
```
you can browse python packages [here](https://pypi.org/)

to see all installed packages:
```
pip list
# OR
pip freeze
```

#### To install from a file 

`pip install -r requirements.txt`

**REMEMBER!**

ALWAYS SPECIFY THE VERSION YOU INSTALL! 

Even if it's the latest version!
